# 🔍 Industrial Defect Detection using Autoencoders

An Unsupervised Deep Learning project for manufacturing quality inspection using Convolutional Autoencoders and reconstruction-error-based anomaly detection.

---

## 📌 Project Overview

Industrial manufacturing environments require continuous quality inspection to identify defective products before they reach customers.

This project implements a **Convolutional Autoencoder** trained only on normal product images. During testing, the model reconstructs the input image and calculates the reconstruction error. Images with high reconstruction error are classified as anomalous or defective.

The system was evaluated on multiple categories from the **MVTec Anomaly Detection Dataset**, including:

* Bottle
* Transistor
* Zipper

---

## 🎯 Objectives

* Learn normal product patterns using Autoencoders.
* Detect anomalies without requiring defect labels during training.
* Visualize defective regions using heatmaps.
* Analyze reconstruction error for industrial quality inspection.

---

## 🧠 Methodology

### Step 1: Training

The Autoencoder is trained only on normal (good) images.

### Step 2: Reconstruction

The model reconstructs the input image through:

Input → Encoder → Latent Space → Decoder → Reconstructed Image

### Step 3: Error Computation

Reconstruction Error is calculated using Mean Squared Error (MSE).

### Step 4: Defect Detection

If reconstruction error exceeds a predefined threshold:

* Defect Detected

Otherwise:

* Normal Product

---

## 🏗️ Model Architecture

```text
Input Image
      ↓
 Convolutional Encoder
      ↓
    Latent Space
      ↓
 Convolutional Decoder
      ↓
Reconstructed Image
      ↓
Reconstruction Error
      ↓
Normal / Defective
```
